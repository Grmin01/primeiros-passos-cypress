# My Cypress Automation

This project utilizes Cypress to automate the testing of a login feature and a user update feature. Cypress is a JavaScrip-based end-to-end testing framework that allows devolopers to write tests and execute them a real browser.

 ## Installation
```bash
npn install
```
> **NOTE:**
>
> It is necessary to have **node** installed

## Running
```bash
#You open the Cypress UI
npx cypress open

# Run the automation by command line
nps cypress run
```
